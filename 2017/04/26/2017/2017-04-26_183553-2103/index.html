查看 github star 排名和 fork 排名

方法来自 stackoverflow：http://stackoverflow.com/questions/19855552/how-to-find-out-the-most-popular-repositories-on-github，可以看到目前 github star 或 fork 最多的项目：

fork 最多的项目

https://github.com/search?o=desc&q=stars:%3E1&s=forks&type=Repositories

star 最多的项目

https://github.com/search?q=stars:%3E1&s=stars&type=Repositories
