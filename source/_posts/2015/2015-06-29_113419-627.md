通过系统提供的函数，我们一样可以查看、修改和删除环境变量的内容，使用使用函数分别时 getenv()、setenv()、unsetenv()，它们的用法如下：
<!-- more -->
```
#include <stdio.h>
#include <stdlib.h>

int main(int argc, char* argv[])
{
    // 打印当前 PATH 环境变量的内容
    printf("PATH=%s\n", getenv("PATH"));
    // 重新设定 PATH 环境变量的内容
    setenv("PATH", "hello", 1);
    // 再次打印被重新设定后的环境变量内容
    printf("PATH=%s\n", getenv("PATH"));
    // 删除环境变量值
    unsetenv("PATH");
    printf("PATH=%s\n", getenv("PATH"));
    return 0;
}
```
